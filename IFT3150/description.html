Description of the Project

Title: Extending Generalization in Reinforcement Learning

Topic of the project:
I decided to fully focus on the concept of generalization using different Atari environments since I already wrote two different CNN policies for different Atari environments.

The goal here is to experiment on three different environments using CNN policy. The agent trains on three different games and it needs to learn the fourth game by itself (a new game that the agent has never seen before) based on its experience.

Programming Language: Python 3.7

Libraries:
- Open AI, stable-baselines3, Pytorch
- rlkit: Library for Reinforcement Learning Algorithms
- doodad: Library for running experiments on different computers
